# Bowling Game Project

## Overview
This project implements a 10-pin Bowling Game score calculator in Python.
Supports strikes, spares, open frames, and 10th frame bonuses.

## Features
- Record and calculate scores for each frame
- Supports strikes and spare bonuses
- Handles 10th frame bonus rolls correctly

## File Structure
- BowlingGameProject/
- bowling_game.py          # Refactored BowlingGame class
- test_bowling_game.py     # Unit tests using pytest
- examples.py              # Example games for verification
- README.md                # Project overview and instructions
- docs/                    # PythonDoc or documentation files

### Requirements
- Python 3.x
- pytest (for running unit tests)

### Install Dependencies
pip install pytest

### Running the Game Examples
python examples.py

### Running Unit Tests
pytest test_bowling_game.py
